# Web Services com Spring Boot e JPA
## Workshop Spring Boot 3 com JPA
### Bem-vindo ao repositório do Workshop Spring Boot 3 com JPA! Este projeto é uma aplicação de exemplo criada para ensinar os conceitos e práticas de desenvolvimento com Spring Boot 3 e Java Persistence API (JPA).
# Sobre o Projeto
### Este projeto é parte de um workshop que visa ensinar os participantes a construir uma aplicação web robusta usando Spring Boot 3 e JPA. O projeto exemplifica as práticas recomendadas para configuração, desenvolvimento e testes de uma aplicação Spring Boot.
# Tecnologias Utilizadas
Java 17
Spring Boot 3
Spring Data JPA
H2 Database (para desenvolvimento e testes)
Maven
